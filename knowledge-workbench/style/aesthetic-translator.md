# 审美翻译器

目标：把“高级感、Claude 味、性冷淡、不土、现代但克制”这类抽象说法，先翻译成可执行的设计判断，再开始做页面、PPT、文档或演示。

## 翻译总原则
- 不要把风格词直接等同于某种流行 UI 皮肤。
- 不要把“高级感”理解成炫技、发光、玻璃或复杂纹理。
- 先判断用户真正想保留的感觉，再判断最容易跑偏成什么。
- 风格翻译优先服务信息主次和讲述节奏，而不是装饰堆叠。

## 默认翻译维度
接到抽象风格词时，至少拆成这 6 维：

### 1. 气质
- 冷静 / 温和 / 锐利 / 可信 / 编辑感 / 制度感 / 未来感 / 松弛感

### 2. 版式
- 大标题还是中密度标题
- 大留白还是紧凑信息面
- 对称还是非对称
- 稳定网格还是节奏变化

### 3. 材料语言
- 纸感
- 书面感
- 金属感
- 磨砂感
- 屏幕感
- 工业感
- 轻界面感

### 4. 信息密度
- 稀疏
- 中密度
- 高密度但秩序强

### 5. 节奏
- 安静平稳
- 前紧后松
- 先抓判断再铺证据
- 连续叙事型

### 6. 禁忌项
- 哪些元素一出现就会土
- 哪些元素会把风格带偏成 AI 模板味

## 高频风格词翻译表

### 1. “像 Claude 一样高级”
- 真正想要：自然、克制、编辑感、不过度设计
- 版式判断：层级清楚但不机械；更像杂志页，不像控制台或仪表盘
- 材料语言：纸感、书面感、轻界面感
- 节奏：安静、顺滑、判断先于装饰
- 禁忌项：玻璃卡、炫光渐变、组件堆叠、为了高级而堆纹理

### 2. “BMW 性冷淡风”
- 真正想要：冷静、精密、低噪音、工业质感
- 版式判断：大块留白、模块少、比例稳、标题强、说明短
- 材料语言：金属感、磨砂感、安静工业感
- 节奏：平稳克制，不突然炸开
- 禁忌项：紫蓝科技渐变、发光描边、玻璃拟态、过多控件

### 3. “学术但不土”
- 真正想要：可信、理性、清楚、轻现代
- 版式判断：图文主次分明，结构强，修饰少
- 材料语言：纸面、细线、浅底、少阴影
- 节奏：先结论，再说明，不拖
- 禁忌项：传统模板蓝、厚重阴影、廉价图标、PPT 组件味

### 4. “现代但克制”
- 真正想要：当代感，但不浮夸
- 版式判断：少量高对比重点，其余部分留给秩序和留白
- 材料语言：干净纯色、低饱和中性色、少量精确强调色
- 节奏：有推进，但不躁动
- 禁忌项：大面积花哨渐变、为了现代感而加复杂动效

### 5. “未来感但不要赛博”
- 真正想要：锋利、清洁、轻技术感，不要黑客味
- 版式判断：结构明确，直线感强，重点稀少但精确
- 材料语言：冷金属、低饱和反光、精准线条
- 节奏：干净、收束、不狂躁
- 禁忌项：霓虹色、矩阵雨、终端皮肤、赛博朋克紫蓝发光

### 6. “像杂志一样”
- 真正想要：编辑感、留白、图文张力、阅读气质
- 版式判断：可以非对称，但不能乱；标题、图片、说明形成节奏
- 材料语言：纸感、印刷感、书面感
- 节奏：有呼吸感，页与页之间有展开关系
- 禁忌项：控制台风、仪表盘风、组件网格平均铺开

### 7. “简洁但不要无聊”
- 真正想要：少，但有判断；克制，但不乏味
- 版式判断：保留少量高对比焦点，不要平均铺开
- 材料语言：中性色底 + 一处主强调
- 节奏：至少有一到两个节奏变化点
- 禁忌项：所有元素一个大小、一个灰度、一个节拍

## 默认反跑偏检查
在真正产出前，默认先问自己：
- 我是不是把“高级”误做成了“炫”
- 我是不是把“克制”误做成了“空”
- 我是不是把“现代”误做成了“科技皮肤”
- 我是不是把“编辑感”误做成了“拼贴混乱”
- 我是不是把“学术”误做成了“模板蓝 + 文字墙”

## 默认输出方式
如果用户只给一个抽象风格词，默认优先输出：
1. 这个词我会如何翻译
2. 最适合的 2 到 3 个方向
3. 每个方向的版式重点
4. 明确禁忌项

不要一上来就直接给最终视觉成品。
