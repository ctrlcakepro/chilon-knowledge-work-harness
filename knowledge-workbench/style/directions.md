# 审美方向库

目标：让设计、PPT、版式、页面和内容包装任务先做风格判断，再做具体方案。

## 1. 学术冷静型
- 关键词：克制、清晰、低噪音、可信
- 适用场景：课程汇报、论文展示、读书报告、研究说明
- 版式重点：清楚层级、大留白、少装饰、标题有力
- 文案气质：稳、准、不过分煽动

## 2. 现代叙事型
- 关键词：流动、节奏感、讲述性、轻视觉张力
- 适用场景：主题展示、项目介绍、带故事线的报告
- 版式重点：先抓主线，再铺信息，页面之间有节奏变化
- 文案气质：有推进感，不死板

## 3. 制度清晰型
- 关键词：秩序、框架、结构感、规范
- 适用场景：政策解读、制度分析、会议材料、说明型文档
- 版式重点：模块边界清楚，信息分区明确，图表与列表配合
- 文案气质：明确、稳健、少修辞

## 4. 汇报说服型
- 关键词：重点前置、判断明确、说服力、记忆点
- 适用场景：课堂展示、竞赛答辩、方案汇报、立场论证
- 版式重点：第一页就给主判断，后续页面为判断服务
- 文案气质：有结论、有方向，不拖沓

## 默认使用方式
1. 先从四个方向里选 2 到 3 个最适合当前任务的方向。
2. 说明各自适用原因。
3. 再决定信息主次、版式节奏和视觉重点。
