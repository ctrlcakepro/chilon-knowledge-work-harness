# Harness Drills

## Slides 回退
检查 slides 能力不可用时，是否仍能给出页序、主旨和讲稿骨架。

## 最新事实核对回退
检查无法确认最新事实时，是否明确标注待核对，而不是假装确认。

## 表格交付回退
检查表格能力不可用时，是否能给出可复制表格或 CSV 结构。

## 正式文档交付回退
检查文档能力不可用时，是否能保留文档结构与版式说明。

## PDF 审阅回退
检查 PDF 审阅能力不可用时，是否说明限制并给出可继续操作的文本审阅路径。

## 文本选择器追问回退
检查关键分叉处是否能用短选项追问。

## 文本选择器实战演练
使用真实请求演练 2 到 4 个选项的收口方式。
