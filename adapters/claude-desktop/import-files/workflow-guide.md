# Workflow Guide

This file gives Claude a lightweight routing map for common knowledge-work tasks.

## 1. Reading and Study Notes

Use this when the user provides articles, textbook chapters, course materials, lecture notes, or PDFs.

Default output order:

1. Topic and material type
2. Main argument or central structure
3. Key concepts
4. Evidence, examples, or important details
5. Study notes or review version
6. Possible exam points or discussion angles when useful

## 2. Knowledge Organization

Use this when the user has scattered notes, fragments, screenshots, copied paragraphs, or mixed materials.

Default output order:

1. Clean categories
2. Main structure
3. Reusable notes
4. Missing or unclear parts
5. Suggested next step

## 3. Comparison and Judgment

Use this when the user asks to compare theories, tools, plans, texts, or arguments.

Default output order:

1. Comparison dimensions
2. Table
3. Short judgment
4. Best use case for each option
5. Caveats

## 4. Paper and Assignment Writing

Use this when the user asks for a paper, assignment answer, short essay, literature review, or course response.

Default output order:

1. Interpret the task
2. Suggest thesis direction
3. Build outline
4. Give key arguments
5. Expand into draft only when requested

## 5. Presentation and Slides

Use this when the user needs a presentation structure, speech plan, classroom report, or slides outline.

Default output order:

1. Audience and time assumption
2. Main message
3. Storyline
4. Slide-by-slide structure
5. Speaker notes when useful

## 6. Text Revision

Use this when the user asks to polish, rewrite, shorten, expand, or make text more professional.

Default output order:

1. Main issue diagnosis
2. Revised version
3. Optional notes on what changed

For short direct rewrite requests, prioritize the revised version instead of long critique.

## 7. Fact Checking

Use this when the user asks whether something is true, current, accurate, or well-supported.

Default output order:

1. State what can be confirmed from provided material
2. Identify what is inference
3. Mark what needs external verification
4. Give a cautious conclusion
